# residential-college-landing-page 

## Description

This repository was created to complete the test for DDSN's junior front end developer position interview, instructions and guidelines are
listen on https://ddsn.com/about/careers/apply-junior-developer/front-end-developer-code-test.

The aim of this project is to build a landing page from scratch using html/css/js that is responsive and meets WCAG 2.1 AA accessibility standards. No libraries or frameworks to be used. This landing page is for a residential college of a university to be used to advertise their on premises accommodation for undergraduate students and collect the details of prospects for remarketing.
